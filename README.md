# IMAGE CLASSIFICATION USING CONVOLUTIONAL NEURAL NETWORKS (CNN)

Built and trained a CNN from scratch using `Pytorch` for classification on a complex and heavy dataset.

---

## DATASET

Food Images (Food-101) by K Scott Mader  
https://www.kaggle.com/datasets/kmader/food41  
  
Train set : `10099`
Test set : `1000`

- Python
- Pytorch
- torchvision
- Scikit-learn
- Matplotlib.pyplot
